#  Budget Insights & Expense Tracker

##  Live Demo
[https://yourproject.vercel.app](https://yourproject.vercel.app)

##  Description
This is a smart expense tracking app built with **React, Vite, and Tailwind CSS**.  
It helps you add income and expenses, view transaction history, and check your balance with a pie chart.  
I built this project to practice React, Redux Toolkit, and responsive UI design.

##  Features
- Add income & expense transactions
- Transaction history with search and filter
- State management using Redux Toolkit
- Responsive UI with TailwindCSS
- Interactive Pie chart for Income vs Expense
- Budget view with category breakdown
- Hover effects, gradients, and smooth animations

##  Project Goals
- Learn and apply Redux Toolkit for state management  
- Practice responsive design with TailwindCSS  
- Deploy a working project on Vercel  
  

## 🛠️ Technologies & Dependencies
- **React** (frontend framework)
- **Vite** (bundler & dev server)
- **Tailwind CSS** (styling)
- **Redux Toolkit** (state management)
- **react-chartjs-2** and **chart.js** (charts & visualization)
- **React Router** (if used for navigation)
